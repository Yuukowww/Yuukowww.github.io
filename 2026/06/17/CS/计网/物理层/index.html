通信基础

基本概念

码元是通信传输的基本单位，通常将若干个bit编码而成。

通信系统的构成可以分为信源、信道和信宿。

信源 - DAC变换器 - 信道 - ADC变换器 - 信宿
                   |
                  噪声

数据传输的方式可以分为串行传输与并行传输

  • 串行传输 -- 网络传输、总线传输
  • 并行传输 -- CPU和内存间的传输为并行传输

按是否调制可以将传输信号分为基带信号与宽带信号

  • 基带信号 -- 信源发出的原始信号
  • 宽带信号 -- 经过调制到高频载波上的信号

信道的传输中有两个重要的概念

  • 波特率 -- 信道内的码元传输速率
  • 比特率 -- 信道内的bit传输速率

根据码元的定义，显然有

比特率=波特率×单个码元的信息量\text{比特率} = \text{波特率}\times \text{单个码元的信息量} 比特率=波特率×单个码元的信息量

信道的容量

信息容量的限制根据两个信息论的基本定理限制: Nyquist定理与 Shannon定理。二者分别在理论采样情况下与含噪声情况下进行信道理论传输速率的上限讨论

Nyquist采样定理

Nyquist 定理说明在信号原始最高频率为WWW时，离散采样频率大于等于2W2W2W 才不会导致混叠现象。最高频率的两倍称为Nyquist 频率

根据Nyquist定理，对于信道频率为WWW 的信道，其最大波特率为2W2W2W, 假设单个码元有V个离散电平表示(即可以编码 log⁡2V\log_2 Vlog2​V 个bit ), 因此理想比特率为

R=2Wlog⁡2VR = 2W\log_2 V R=2Wlog2​V

Shannon–Hartley 定理

Shannon定理从信息熵的角度刻画含高斯噪声信道的传输上限

考虑信道频率带宽为WWW, 信号平均功率SSS， 噪声平均功率NNN 的信道，信道的理想比特率为

R=Wlog⁡2(1+SN)R = W\log_2 \left(1+\frac{S}{N}\right) R=Wlog2​(1+NS​)

其中信噪比定义为η=SN\displaystyle \eta = \frac{S}{N}η=NS​， 使用对数单位分贝表示时, 满足 ηdB=10log⁡10SN\eta_{dB} = 10\log_{10}\dfrac{S}{N}ηdB​=10log10​NS​

Nyquist定理与Shannon-Hartley 定理共同给出了AWGN信道的理论信道容量上限，即单个码元的二进制位上限

C=12log⁡2(1+SN)C = \frac{1}{2}\log_2\left(1+\frac{S}{N}\right) C=21​log2​(1+NS​)

对应的bit上限为

bit=⌊1+SN⌋\text{bit} =\left\lfloor \sqrt{1+\frac{S}{N}} \right\rfloor bit=⌊1+NS​​⌋

编码与调制

将数据转换为数字信号的过程称为，将数据载波到模拟信号过程称为调制。

数字数据编码为数字信号

编码的核心在于如何使用电平跳变表示两个不同的状态，即二进制0 与1

常见编码有

  • 归零编码(RZ) -- 1表示高电平跳变到0；0表示低电平跳变到0，在半周期位置实现跳变，用于时钟基准标记。
  • 非归零编码(NRZ) -- 1表示高电平;0表示低电平, 半周期位置不进行跳变，因此无法进行时钟标记；高/低电平覆盖整个周期，因此编码效率最高
  • 反向非归零编码(NRZI) -- 1表示周期起始沿不跳变；0表示起始延长跳变。跳变本身作为脉冲可以进行辅助时钟的校准，且编码效率较高。USB2.0使用NRZI编码，NRZI 会面临如长时间传输1导致的失去位边界问题，导致时钟信号漂移与数据分段错乱。
  • 曼切斯特编码 -- 使用下降沿表示1，上升沿表示0；半周期时跳变以实现时钟标记
  • 差分曼切斯特编码 -- 根据当前码元决定当前周期起始沿是否跳变，如果为1则不跳变，为0则跳变。半周期发生跳变以实现时钟基准标记。

半周期跳变实现的时钟基准标记与半个周期的归零引起的带宽占用总是冲突的。

数字数据调制为模拟信号

实现的核心为在模拟信号中体现两个状态。常见的实现方式分为

  • 幅移键控(ASK) -- 0表示没有幅度的无载波输出，1表示存在幅度的载波输出。使用一个开关器就能实现，但是抗干扰能力较差，对于加性高斯噪声的正向偏移没有抵抗能力
  • 频移键控(FSK) -- 设置两个频率状态 f0f_0f0​, f1f_1f1​, 0时输出频率为f0f_0f0​的模拟信号，1时输出频率为f1f_1f1​的模拟信号
  • 相移键控(PSK) -- 正弦波的000 相位半周期和π\piπ 相位半周期反向，基于此，0 输出为 000 相位半周期, 1 输出为π\piπ 相位半周期
  • 差分相移键控(DPSK) -- 基于前一码元分析是否相移，可以通过0标定相移也可以通过1标定相移
  • 正交幅度调制(QAM)

QAM同时调制振幅与相位，在一个二维平面分布信息点，横轴代表余弦载波的幅度，纵轴代表正弦载波的幅度，而标准正弦函数与余弦函数是正交的，因此二维幅度编址是可行的

物理层

信号传输介质

有线介质

  • 双绞线 -- 最常见的网线就是双绞线，由两条相互绝缘的铜导线绞合而成，可以有效减少相邻导线的电磁干扰。外加金属屏蔽层的双绞线称为屏蔽双绞线(STP)，反之为非屏蔽双绞线(UTP)
  • 同轴电缆 -- 内导体+绝缘层+外导体屏蔽层+绝缘保护套层
  • 光纤 -- 光纤分为多模光纤与单模光纤。多模光纤基于光的全反射原理，损耗较大，适合短距离通信；单模光纤基于超细纤径介质内光的直线性，当光纤直径接近波长时，几乎无反射损失，适合长距离通信。

无线介质

  • 无线电波 -- 穿透能力强，无指向性
  • 微波 -- 高频、带宽大、通信容量高，需要指向性与中继接力
  • 红外线
  • 激光
  • 卫星通信

物理层设备

中继器

将信号整形、放大与转发，通过信号再生的方式放大处理信号

集线器

集线器是多端口的中继器，多端同时向集线器传输信号会导致冲突，所有相关帧都会失效。对于单输入的信息，其它输出端口会广播输入的信息，属于共享型设备。

集线器网络的拓扑结构是星形网络，但是逻辑连接是总线形网络，和总线网络的信道资源竞争、广播等特征一致。

Nyquist 采样与离散化

由 Nyquist 采样定理，带宽为 WWW 的信号可由采样率 fs=2Wf_s = 2Wfs​=2W 的离散样本无失真表示。因此每秒可传输 2W2W2W 个独立的实值样本，AWGN 信道的每次信道使用等价为离散时间模型

Yi=Xi+Zi,Zi∼N(0,N),i=1,2,…Y_i = X_i + Z_i, \qquad Z_i \sim \mathcal{N}(0, N), \quad i = 1, 2, \dots Yi​=Xi​+Zi​,Zi​∼N(0,N),i=1,2,…

其中 N=N0WN = N_0 WN=N0​W 为噪声功率，E[Xi2]≤S\mathbb{E}[X_i^2] \leq SE[Xi2​]≤S 为信号功率。

互信息与微分熵

对于实数随机变量，互信息可表示为微分熵的差：

I(X;Y)=h(Y)−h(Y∣X)I(X;Y) = h(Y) - h(Y \mid X) I(X;Y)=h(Y)−h(Y∣X)

由于 ZZZ 与 XXX 独立，h(Y∣X)=h(X+Z∣X)=h(Z∣X)=h(Z)h(Y \mid X) = h(X + Z \mid X) = h(Z \mid X) = h(Z)h(Y∣X)=h(X+Z∣X)=h(Z∣X)=h(Z)，因此

I(X;Y)=h(Y)−h(Z)I(X;Y) = h(Y) - h(Z) I(X;Y)=h(Y)−h(Z)

噪声 Z∼N(0,N)Z \sim \mathcal{N}(0, N)Z∼N(0,N) 的微分熵为

h(Z)=12log⁡2(2πeN)h(Z) = \frac{1}{2} \log_2\bigl(2\pi e N\bigr) h(Z)=21​log2​(2πeN)

在功率约束 E[X2]≤S\mathbb{E}[X^2] \leq SE[X2]≤S 下，由于 XXX 与 ZZZ 独立，输出 YYY 的方差为

Var(Y)=Var(X)+Var(Z)≤S+N\mathrm{Var}(Y) = \mathrm{Var}(X) + \mathrm{Var}(Z) \leq S + N Var(Y)=Var(X)+Var(Z)≤S+N

高斯输入的最优性

引理（最大熵定理）：对于给定方差 σ2\sigma^2σ2 的所有连续分布，均值为零的高斯分布 N(0,σ2)\mathcal{N}(0, \sigma^2)N(0,σ2) 使微分熵达到最大：

h(Y)≤12log⁡2(2πe σ2)h(Y) \leq \frac{1}{2}\log_2\bigl(2\pi e \,\sigma^2\bigr) h(Y)≤21​log2​(2πeσ2)

等号成立当且仅当 Y∼N(0,σ2)Y \sim \mathcal{N}(0, \sigma^2)Y∼N(0,σ2)。

因此，当输入 X∼N(0,S)X \sim \mathcal{N}(0, S)X∼N(0,S) 时，Y=X+Z∼N(0,S+N)Y = X + Z \sim \mathcal{N}(0, S + N)Y=X+Z∼N(0,S+N)，微分熵达到上界

h(Y)≤12log⁡2(2πe(S+N))h(Y) \leq \frac{1}{2} \log_2\bigl(2\pi e (S + N)\bigr) h(Y)≤21​log2​(2πe(S+N))

代入互信息表达式：

I(X;Y)=h(Y)−h(Z)≤12log⁡2(2πe(S+N))−12log⁡2(2πeN)=12log⁡2 ⁣(S+NN)=12log⁡2 ⁣(1+SN)\begin{aligned} I(X;Y) &= h(Y) - h(Z) \\[4pt] &\leq \frac{1}{2} \log_2\bigl(2\pi e (S + N)\bigr) - \frac{1}{2} \log_2\bigl(2\pi e N\bigr) \\[4pt] &= \frac{1}{2} \log_2\!\left(\frac{S + N}{N}\right) \\[4pt] &= \frac{1}{2} \log_2\!\left(1 + \frac{S}{N}\right) \end{aligned} I(X;Y)​=h(Y)−h(Z)≤21​log2​(2πe(S+N))−21​log2​(2πeN)=21​log2​(NS+N​)=21​log2​(1+NS​)​

信道容量公式

乘以 Nyquist 速率 2W2W2W，得到 Shannon–Hartley 信道容量公式：

C=Wlog⁡2 ⁣(1+SN)(bit/s)\boxed{C = W \log_2\!\left(1 + \frac{S}{N}\right) \qquad \text{(bit/s)}} C=Wlog2​(1+NS​)(bit/s)​

或以分贝形式的 SNR 表示（ηdB=10log⁡10(S/N)\eta_{\text{dB}} = 10 \log_{10}(S/N)ηdB​=10log10​(S/N)）：

C=Wlog⁡2 ⁣(1+10ηdB/10)C = W \log_2\!\left(1 + 10^{\eta_{\text{dB}}/10}\right) C=Wlog2​(1+10ηdB​/10)

与 Nyquist 公式的联系

在 AWGN 信道中，Nyquist 定理给出无噪声条件下的码元速率上限 2W2W2W，Shannon 定理进一步给出每个码元所能承载的比特数上限 12log⁡2(1+S/N)\frac{1}{2}\log_2(1 + S/N)21​log2​(1+S/N)。二者共同决定 AWGN 信道的理论容量：

C=2W⏟Nyquist: 最大波特率  ×  12log⁡2 ⁣(1+SN)⏟Shannon: 每码元最大 bit 数C = \underbrace{2W}_{\text{Nyquist: 最大波特率}} \;\times\; \underbrace{\frac{1}{2}\log_2\!\left(1 + \frac{S}{N}\right)}_{\text{Shannon: 每码元最大 bit 数}} C=Nyquist: 最大波特率2W​​×Shannon: 每码元最大 bit 数21​log2​(1+NS​)​​

因此单个码元的二进制位数上限为

Cper symbol=12log⁡2 ⁣(1+SN)C_{\text{per symbol}} = \frac{1}{2}\log_2\!\left(1 + \frac{S}{N}\right) Cper symbol​=21​log2​(1+NS​)

对应的理想调制电平数上限为

Vmax⁡=⌊1+SN⌋V_{\max} = \left\lfloor \sqrt{1 + \frac{S}{N}} \right\rfloor Vmax​=⌊1+NS​​⌋
