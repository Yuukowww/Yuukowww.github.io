# Repository Instructions

## Commit Policy

- When Codex creates a git commit in this repository, include this trailer at the end of the commit message:

  ```text
  Co-authored-by: Codex <noreply@openai.com>
  ```

- Keep the trailer as the final non-empty line unless another required trailer is explicitly requested.
- Stage only files that are part of the user's requested change.
- Push only when the user explicitly asks for a push.
