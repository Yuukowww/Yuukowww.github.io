# 从 tabaco 到“烟草”和「煙草（たばこ）」

如果只看字形，中文“烟草”和日语「煙草」像是同一个汉字词在两地保留下来；但如果把声音也放进来，它们其实是两条不同的词汇史在同一个字形上交会：

- 中文的“烟草”是把外来物重新命名为一个透明的汉语复合词。“淡巴菰 / 淡巴枯 / 淡把姑”等早期音译并没有音变成 yancao，而是被“烟 + 草”这个语义更清楚的名称替换。
- 日语的「たばこ」则是南蛮贸易时期进入日本的外来词，通常认为来自葡萄牙语或西班牙语 tabaco。后来写作「煙草 / 莨」等汉字，但读音保留外来词本身，所以不是逐字读成「けむり + くさ」，也不是按音读读成「えんそう」。

换句话说：中日同用“烟草 / 煙草”这组字，不等于它们继承了同一个“汉语读音”。中文把词改造成了汉语内部的形态；日语把汉字当作意义标签，贴在已经流通的外来读音上。

## 一条大航海时代的传播线

烟草原产美洲。欧洲人接触它之后，西班牙语 tabaco、葡萄牙语 tabaco 一类名称随殖民贸易扩散。Online Etymology Dictionary 把英语 tobacco 追到西班牙语 tabaco，并指出该词通常被认为至少部分来自加勒比阿拉瓦克语族、可能是泰诺语中的烟叶卷或吸烟管名称。

东亚这边的关键中转站是菲律宾与南海贸易。16 世纪后期，西班牙人在菲律宾经营马尼拉-阿卡普尔科帆船贸易，美洲作物和美洲白银一起进入亚洲贸易网。日本“たばこと塩の博物館”在介绍大航海时代时也把烟草列在从新大陆经欧洲扩散到世界各地的物产之中。进入日本的路径常被放在南蛮贸易背景下讨论：1543 年葡萄牙人漂到种子岛以后，九州港口逐渐卷入日欧贸易，烟草也作为南蛮渡来的珍物进入日本。

日本方面，确切早期记录集中出现在 16 世纪末到 17 世纪初。JTI 年表列出，1601 年方济各会士向德川家康献上烟草叶和烟草种子；1605 年左右已有关于吸烟、买卖、耕作的禁令。博物馆的“江户时代的たばこ文化”也说，日本在 16 世纪末以后传入烟草，吸烟习惯扩散，国内种植开始，幕府随即因为风纪、火灾、粮食生产等原因多次禁烟。

中国方面，传入时间也大致在明末万历年间。烟草约在 16 世纪末、明万历年间传入中国，最早译音叫“淡巴枯”，明末改用“烟草”名称。日本 y-history 的“タバコ”条目在“中國と日本のタバコ”部分整理了几种传入路线：菲律宾到福建、东南亚海路到广东、日本经朝鲜到辽东、印度支那内陆到云南等；其中史料上以福建、广东经路较多，并提到万历末年“淡把姑”传入福建的说法。

这说明：烟草进入东亚不是通过一条单纯的“中文词传给日文”路线，而是伴随海商、传教士、南蛮贸易、菲律宾贸易网、东南亚贸易网，在多个港口同时扩散。物和名字都带着外来性，但每种语言给它安家的方式不同。

## 中文：从音译名到语义名

明清材料中，烟草有一串不稳定的名字：淡巴菰、淡巴枯、淡把姑、淡巴菇、担不归、金丝烟、芬草等。这些名称可以分成两类：

一类是音译，围绕 tabaco 的声音做汉字拟音。例如“淡巴菰 / 淡巴枯 / 淡把姑”都明显在模拟 ta-ba-co 一类音节。由于它从吕宋、福建、广东、东南亚等多口岸进入，又经过不同方言和书写者记录，字形自然不会统一。

另一类是语义命名。“金丝烟”偏向加工形态：烟叶切成细丝；“烟草 / 菸草”则抓住最核心的两个可见特征：它是草本植物，使用方式产生烟。这个名字不需要知道它从哪里来，也不需要知道 tabaco 怎么发音，第一次看见也大概能猜到它是什么。

“淡巴菰”没有发展成现代通用名，原因不在于它读音不够接近，而在于它不适合成为普通汉语词：

1. 音译字形不稳定。“菰、枯、姑、菇”等只是借音，难以形成统一规范。
2. 语义负担太低。普通读者看到“淡巴菰”无法从字面理解其用途；看到“烟草”却能立刻建立类别。
3. 物品本土化以后，外来名的必要性下降。烟草很快在福建、广东、江南、北方军镇等地流行并种植，已不再只是“番货”，而是日常消费品和经济作物。
4. 汉语构词偏好双音节、可组合、可派生的语素结构。“烟草”可以自然派生出烟叶、烟丝、烟袋、烟枪、抽烟、禁烟等词群；“淡巴菰”则很难进入这个网络。

所以中文这里发生的不是“音译词音变为原始汉语读音”，而是一次词汇替换：早期外来音译词被本土语义复合词取代。现代汉语读 yancao，是因为现代词项已经是“烟 + 草”，不是因为 ta-ba-co 在汉语音系里逐步演化出了 yan-cao。

## 日语：外来读音先固定，汉字后来表义

日语侧的核心恰好相反：读音「たばこ」先作为外来词固定下来，然后可以写成「煙草」。

Weblio 收录的《デジタル大辞泉》词条写作「タバコ【(ポルトガル)tabaco】【煙草／莨】」，并标注读法为「たばこ」。同一页面还列出「煙草」可有「タバコ、ケムリグサ、ケブリグサ、エンソウ」等读法；这点很有意思，因为它说明「けむりぐさ」和「えんそう」并非完全不可想象，而是没有成为现代普通词的默认读音。

原因在日语文字系统本身。日语中的汉字不只对应一种读法：

- 音读：来自不同历史层次的汉字音，如「煙」可读 えん，「草」可读 そう。
- 训读：用汉字表示日本固有词，如「煙」读 けむり，「草」读 くさ。
- 熟字训 / 义训 / 当て字一类整词读法：字形主要提示意义，整组字读作一个已经存在的词，如「大人（おとな）」「昨日（きのう）」等；外来词也常被配上汉字表记。

「煙草（たばこ）」就属于这种“字形表义、读音走词项”的类型。它不是把「煙」读作 たば、「草」读作 こ；而是整组「煙草」被用来写已经存在的外来词 tabako。汉字在这里像一个意义说明：这种东西是会产生烟的草本/草叶；读音则继续沿用贸易和日常口语中的外来名。

这也解释了为什么日语没有把它稳定读成「けむりくさ」。如果日本先独立观察到一种“会冒烟的草”，再用固有词造名，「けむりぐさ」可能会有机会。但实际历史中，烟草连同吸烟方式、贸易物品身份、南蛮文化气味一起进入日本，名字「たばこ」本身就是新奇物的标签。等到汉字「煙草」贴上去时，口语词已经站稳了。

至于音读「えんそう」，它也缺少胜出的理由。音读通常更适合从汉文、佛典、制度、学术语汇中进入的汉字词；烟草在日本不是作为一个汉文概念先被读书人引入，而是作为南蛮贸易商品、吸烟风俗和种植作物进入民间生活。传播路径是“物和口头名先到”，不是“汉字词先到”。

## 为什么两边用了相同的汉字？

“烟 / 煙 + 草”是一个非常自然的语义组合：它既说明使用时有烟，又说明原料是草本植物或草叶。对于汉字文化圈的书写者来说，这是近乎显而易见的命名方案。

但相同字形在中日两种语言中的功能不同：

中文里，汉字直接参与词的读音。写成“烟草”，就会按汉语读作 yancao。若要保留 tabaco 的声音，就必须继续写“淡巴菰”这类音译字。两套写法竞争时，透明的语义词胜出。

日语里，汉字和读音的关系更松。一个汉字词可以有音读、训读，也可以有整词读法；必要时还可以用假名标读音。写成「煙草」并不强迫它读成「えんそう」或「けむりくさ」。因此日语能够同时保留外来声音「たばこ」和汉字语义「煙草」。

这就是最关键的分叉：

- 中文：外来声音让位于本土语素，词义透明化。
- 日语：外来声音保留，汉字承担意义标注。


早期近代东亚面对外来物时，常有三种策略：

1. 直接音译：保留声音，如“淡巴菰”。
2. 意译或语义重命名：保留概念，如“烟草”。
3. 声音和汉字分工：日语常见，读外来词，写表义汉字，如「煙草（たばこ）」。

中文并非不能音译，日语也并非不能意译；差别在于哪个方案在具体社会环境中更容易被日常使用、书写规范、商业传播和文学记录共同固定下来。

烟草进入中国后，迅速被纳入农业、税收、禁令、医药笔记、日常交际和消费文化；“烟草”这种普通汉语词最方便扩散。烟草进入日本后，「たばこ」作为南蛮外来物的口头名称已经强势流通，汉字「煙草」没有替换它的声音，只是给它一个可读可写、意义清楚的外衣。

## 小结

中文“烟草”和日语「煙草（たばこ）」看起来像同源词，实际上更像“同字异史”：

中文的历史是：tabaco 的音译名进入明末文献，随后被“烟草”这个本土语义复合词取代。现代读音 yancao 来自“烟”和“草”两个汉语语素。

日语的历史是：tabaco 作为南蛮外来词「たばこ」进入口语和生活文化，后来写作「煙草」。现代读音 tabako 保留的是外来词的声音，汉字只是把“会产生烟的草叶”这个意义写出来。

所以，问题的答案不在“为什么日本没有保留煙（けむり）草（くさ）”，而在于：日本从一开始保留下来的就不是“煙 + 草”的组合读法，而是一个外来词；「煙草」只是它的汉字表记。中文则相反，最终通行的不是外来音译，而是汉语内部重新命名后的“烟草”。
