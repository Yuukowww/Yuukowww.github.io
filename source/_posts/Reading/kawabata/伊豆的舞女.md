《伊豆的舞女》并不是我看过的第一本川端康诚的书，相比《雪国》、《千纸鹤》等刻作品刻画的日本压抑严肃社会背景下的扭曲的爱情，《伊豆的舞女》中的萍水相逢间平淡而朴素的好感与爱恋，却如山间的野花纯真，也如野花般易逝。

::: default subtitle no-icon
在黑暗中，少年的体温温暖着我。我任凭泪泉涌流，我的头脑恍如变成一池清水，一滴滴溢了出来，后来什么也没有留下，顿时觉得舒畅了。
:::


熏子是巡回的艺人，是舞女，"我"是独自在伊豆旅行的学生。在茶馆与舞女一行相遇

::: default subtitle no-icon
舞女看上去约莫十七岁光景。她梳着一个我叫不上名字的大发髻，发型古雅而奇特。这种发式，把她那严肃的鹅蛋形脸庞衬托得更加玲珑小巧，十分匀称，真是美极了。令人感到她活像小说里的姑娘画像，头发特别丰厚
:::

"我"在见到舞女的第一眼，心底就漾起对舞女淡淡的、朴素的喜爱，这样的好感驱使着"我"在巡回艺人行将出发时，焦急地追寻着熏子的踪迹，扰乱了"我"的心弦。


::: default subtitle no-icon
不到一小时的功夫，传来了巡回演出艺人整装出发的声响。我再也坐不住了。不过，只是内心纷乱如麻，却没有勇气站出来。我心想: 虽说她们长期旅行走惯了路，但毕竟还是女人，就是让她们先走一二公里，我跑步也能赶上。我身在炉边，心却是焦灼万分。尽管如此，她们不在身旁，我反而得到了解脱，开始胡思乱想。
:::
